Capítulo 2. Requerimientos

Esta aplicación se puede instalar en equipos con sistemas operativos Linux, Windows y Mac OSX.

  • En cuanto a gestores de bases de datos, si utilizamos SQLITE no será necesario instalar nada adicional.

    En cambio si queremos utilizar PostgreSQL o MySQL será necesario instalar en un ordenador el servidor de la base de datos correspondiente.

  • KEME utiliza Látex para la elaboración de algunos informes. En OS-X se recomienda el paquete Mactex; para Windows podemos instalarnos Miktex. Ambas distribuciones son opensource.
