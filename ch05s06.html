El paquete Latex

Para comprobar si tenemos instalado el paquete Latex, podremos introducir desde la línea de comandos la orden látex, si éste programa se encuentra disponible, obtendremos una respuesta parecida a la siguiente:

								pepe@MORTADELO:~latex
								This is Tex, Version 3.14159 (Web2C 7.4.5)
								**q

Introduciremos la letra “q” y pulsaremos intro para salir del programa.
